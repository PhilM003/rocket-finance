# Rocket Finance

A beautiful and intuitive Flutter finance management app with a space-themed design.

## Features

- Dashboard Overview with balance, income, and expenses
- Transaction Management with categorization
- Expense Analytics with pie charts
- Blast Off Animation effects
- Bank Account Tracking
- Local Data Storage using SQLite
- CSV Export functionality

## Tech Stack

- Flutter 3.x
- BLoC Pattern for state management
- SQLite for data persistence
- Material Design 3 with dark theme
- fl_chart for visualizations

## Project Structure

- lib/core/ - Theme and utilities
- lib/data/ - Models, data sources, and repositories
- lib/presentation/ - BLoC, screens, and widgets
- lib/main.dart - Application entry point

## Getting Started

```bash
flutter pub get
flutter run
```
